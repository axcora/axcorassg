## 📦 Bundle Sizes

Quick Start
```bash
npm install axcora 
or 
npm install axcora -g
axcora init myblog 
cd myblog 
npm install
axcora dev 
open http://localhost:3000
axcora build
upload your public folder into your host
axcora serve
```

Clone Starter Project
```bash
git clone https://github.com/mesinkasir/axcora-starter.git
cd axcora-starter
npm install
npm run dev
open http://localhost:3000
```
