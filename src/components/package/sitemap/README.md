### Axcora sitemap

```
npm i axcora-sitemap
```

import in to your frontmatter markdown or on axcora templating
```
---
css:
  theme: 'essentials'
  components:
    - sitemap
---
{% axcora-sitemap /%}
```

After including the component, you can use it both in Markdown articles and directly within your templating files.
