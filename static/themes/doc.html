Customization

Overriding Theme Variables
:root {
  --p500: #009688; /* Custom primary color */
  --radius-lg: 1.5rem; /* Larger radius for modern cards */
}

Place your custom variable overrides in a <style> tag before including the main CSS file.

Support

FAQ
  • How do I add a new component?
    Just follow the documented patterns and use the utility classes!
  • How do I switch to dark mode?
    Click the theme toggle button in the navbar. Dark mode works for all components.
  • Is this framework mobile-friendly?
    Yes! All layouts and components are responsive by default.
  • How do I customize the colors?
    Override the CSS variables in your project as shown above.
